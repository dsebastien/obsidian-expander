# Release Notes

## 1.0.0

Initial release.

### Features

- Variable replacement using HTML comment markers
- Four update modes: auto, manual, once, once-and-eject
- Dynamic values with function expressions
- Date functions: `now()`, `today()`, `format()`
- String functions: `lower()`, `upper()`, `trim()`, `replace()`
- Folder filtering (include and exclude lists)
- Commands: Replace current note, Replace vault, Insert expansion
- Editor decorations with mode badges and refresh buttons
- Settings UI for managing replacements
